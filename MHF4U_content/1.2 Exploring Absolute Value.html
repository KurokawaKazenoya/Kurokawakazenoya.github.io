Absolute Value

The function above is the absolute value function.

On a number line, this function describes the distance of any number x from the origin.

Since distance is always positive, the absolute value changes any number to a positive number.

For example:  

  • |-2|= 2
  • |-3+2|=1
  • |-2+4+9|=11
