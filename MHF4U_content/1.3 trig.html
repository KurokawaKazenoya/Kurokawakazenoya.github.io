Properties of Graphs and Sketching Graphs of Functions

Properties of Graphs

Functions can be categorized based on their graphical characteristics:

  • Domain and Range 
  • Intervals of increase and decrease
  • x-intercepts and y-intercepts 
  • Symmetry (even, odd or neither)
  • Continuity and discontinuity 
  • End behavior

Domain and Range

  • The domain of a function is the complete set of possible values of the independent variable (normally x).
  • The range of a function is the complete set of all possible resulting values of the dependent variable (normally y).

Intervals of Increase and Decrease

  • The interval(s) within a function’s domain, where the y-values of the function get larger, moving from left to right, are called the interval(s) of increase.
  • The interval(s) within a function’s domain, where the y-values of the function get smaller, moving from left to right, are called the interval(s) of decrease.

Symmetry

  • Any function that has rotational symmetry about the origin is called an odd function.  Algebraically, all odd functions have the property f(-x)= - f(x).
  • Any function that is symmetric about the y-axis is called an even function.  Algebraically, all even functions have the property f(-x) = f(x).
  • A function can be even, odd or neither.  In other words, it does not have to be even or odd.

Continuity and Discontinuity

  • Any function that does not contain any holes or breaks over its entire domain is called a continuous function.
  • A break in the graph of a function is called a point of discontinuity.

End Behavior

  • As the name suggests, end behavior describes what happens to y for large negative or large positive values of x in a function.
  • For example, in y = x2 we have the following end behavior:
    • As x increases to large positive values, y increases to large positive values.
    • As x decreases to large negative values, y increases to large positive values.
