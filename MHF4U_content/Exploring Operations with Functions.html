Exploring Operations with Functions

Operations with Functions

If two functions have domains that overlap, they can be added, subtracted, or multiplied to create a new function on that shared domain.

  • Adding:  g(x) = f(x) + h(x)
  • Subtracted:  g(x) = f(x) - h(x)
  • Multiplied:  g(x) = f(x)g(x)

Two functions can be added, subtracted, or multiplied graphically by adding, subtracting, or multiplying the values of the dependent variable for identical values of the independent variable.

 

Two functions can be added, subtracted, or multiplied algebraically by adding, subtracting, or multiplying the expressions of the dependent variable and then simplifying.
