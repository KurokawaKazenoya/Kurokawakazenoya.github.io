Combining Two Functions: Products and Exploring Quotients of Functions

Selecting a strategy to combine function by addition and subtraction

Investigate the product of the functions f(x) = 2-x + 3 and g(x) = sin (2πx)

Using Spreadsheet

In a spreadsheet, enter some values of x in column A, and enter the formulas for f, g, and f ✕ g in columns B, C, and D, respectively.

The values in the table have been rounded to two decimal places.

Looking at each row of the table, for any given value of x,

the function value of (f ✕ g)(x) is represented by f(x) ✕ g(x)

This makes sense since the new function is created by multiplying the original functions together.

Using Manual Graphing

Plotting the ordered pairs (x, (f ✕ g)(x)) results in the graph of the dampened sine wave. This means that the graph of f ✕ g can be obtained from the graphs of functions f and g by multiplying corresponding y-coordinates.
